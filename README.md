## 개발환경
* Spring boot 1.5.8.RELEASE
* Java 8
* JPA & H2
* lombok
* maven

## 프로젝트 실행환경

* Lombok이 반드시 설치 되있어야 합니다.
  - [Eclipse 설치 : [lombok] eclipse(STS)에 lombok(롬복) 설치](http://countryxide.tistory.com/16)
  - [Intell J 설치 : [Intellij] lombok 사용하기](http://blog.woniper.net/229)

```
$ mvn spring-boot:run
```
